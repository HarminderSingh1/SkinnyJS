skinny.js

Low calorie jQuery plugins

skinny.js is a collection of wicked useful, granular jQuery plugins with minimal dependencies. You don't have to buy into any particular framework or coding style. Take what you need; cut the calories.

Get skinny.js

Tiny, focused libraries

Get querystring, URL, and CSS string parsing, cross-browser postMessage(), fast coordinates, temporary event disabling, HTML encoding, custom events, and more...

UI widgets without the fat

Get modal dialogs and menus that are responsive, powerful, and lightweight, and don't require a gigantic framework.

Browser compatibility

skinny.js libraries are tested for all modern browsers and devices, down to IE7.
